ShapeworksStudio (https://www.sci.utah.edu/software/shapeworks.html) is a free shape analysis tool developed by the NIH Center for Integrative Biomedical Computing at the University of Utah Scientific Computing and Imaging (SCI) Institute.

## Installation

Check the Platform Specifications for system requirements.

Installers are provided for Windows and Mac OS X. Linux users need to build ShapeworksStudio from source.
